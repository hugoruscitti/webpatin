Patín

Una herramienta para aprender y construir programando.


¿En qué consiste?

Patín es una herramienta para acercar la programación a los más jóvenes. La razón por la que hacemos esto es porque programar computadoras es fácil, y hay un enorme potencial por descubrir.

Patín logra simplificar el desarrollo de programas tales como videojuegos y aplicaciones de escritorio, ofreciendo un entorno simple, asistido con documentación didáctica, fácil de leer y con muchos ejemplos.

El lenguaje que usamos en Patín es Python, y junto a él están las bibliotecas Qt y pilas.

Libre y comunitario

Patín es software libre, así que podés compartirlo con quien quieras, hacer copias, modificarlo, etc. Además está desarrollado con espíritu comunitario, para que todos podamos añadir mejoras y ser parte del equipo de desarrollo.

Instalación

Para instalar Patín en un sistema Linux puedes bajar el código, o bien, ejecutar este comando: 

sudo pip install patin
